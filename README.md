# Site Blog

**Статический сайт-блог**  
Проект представляет собой вёрстку блога с использованием нативных веб-технологий. Создан в учебных целях для отработки навыков HTML, CSS и JavaScript.

## О проекте
Это классический многостраничный или одностраничный сайт-блог.  
Исходя из структуры репозитория, в проекте используются:

- **HTML** — разметка страниц (файл `index.html` в корне).
- **CSS** — стилизация (папка `styles/`).
- **JavaScript** — базовая интерактивность (папка `js/`).
- **Изображения** — контентные и фоновые картинки (папка `img/`).

## Технологии
- **HTML5** — семантическая вёрстка.
- **CSS3** — стили, адаптивность, сетки.
- **JavaScript (ES5/ES6)** — обработка событий, динамика.

## Структура проекта
 ```
site-blog/
├── index.html          # Главная страница
├── img/                # Изображения для блога
├── js/                 # Скрипты
│   └── main.js         # Основной JavaScript-файл (предположительно)
├── styles/             # Стили
│   └── style.css       # Основная таблица стилей
└── README.md           # Описание проекта
```
